[1.3.0] - 2026-01-06​

✨ 新增​

  • 产品与功能：新增《产品与功能命名规范》，涵盖大写名称、功能名称、缩写与简化。

📝 优化​

  • 排版规范：统一全站禁止名词动词化
  • 视觉优化：为文档进行编号，建立文件夹管理文档

🐞 修复​

  • 去除了《风险规避准则》的无效空格。

[1.2.0] - 2026-01-05​

✨ 新增​

  • 全球视野准则：新增《面向国际受众的写作准则》，涵盖 i18n 核心概念与翻译友好性规范。
  • 决策流程图：为全球化发布增加了 Mermaid 逻辑决策流程。

📝 优化​

  • 排版规范：统一全站中文全角括号 （ ） 的使用规范。
  • 视觉优化：为表格中的 that、then 等辅助词增加了加粗与代码块标注。

🐞 修复​

  • 修正了《风险规避准则》中第 2.4 节的陈旧链接。
  • 修复了 VS Code Webview 预览报错导致的排版显示异常。

[1.1.0] - 2026-01-4​

✨ 新增​

  • 无障碍准则：发布《无障碍写作清单》，支持屏幕阅读器优化。
  • 风险规避准则：确立“禁止预告未来功能”的核心禁令。

📝 变更​

  • 重构了侧边栏层级，将“基础规范”与“进阶合规”进行逻辑分组。

[1.0.0] - 2026-01-01​

  • 初始版本发布：建立技术文档基础设施架构。
